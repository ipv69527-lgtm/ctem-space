# 功能介绍

CTEM Platform 是持续威胁暴露管理平台，覆盖资产接入、单位管理、漏洞管理、风险态势、报表输出和运维验收。

## 资产与单位

- 单位创建、编辑、删除。
- 单位 IP 范围管理。
- 资产按单位归属。
- 资产列表筛选：名称、IP、单位、类型、风险、端口、服务、位置、漏洞状态。
- 资产列配置：经纬度、厂商、品牌、型号、原始记录等可选显示。
- 资产详情画像：基础信息、暴露面、指纹定位、漏洞关联、变更记录。
- 资产人工编辑：可修正未拉取到的名称、IP、MAC、系统、端口、服务、位置、运营商等字段。

## 数据同步

- 支持 RaySpace/Space 接入配置。
- 支持连接测试。
- 支持按单位手动同步。
- 支持同步失败重试。
- 支持自动同步策略。
- 同步任务记录资产拉取数、入库资产数、入库漏洞数和错误详情。

## 漏洞管理

- 漏洞列表、筛选和详情。
- CVE、CVSS、等级、描述、修复建议。
- 影响资产关联。
- 状态流转：待确认、待整改、整改中、待复测、已修复、误报、接受风险。
- 处置备注和更新时间。

## 态势展示

- 全局态势首页。
- 单位风险排行。
- 区域态势地图。
- 大屏展示。
- 资产经纬度和区域风险展示。

## 报表与模板

- 支持生成 Word、Excel、HTML 报表。
- 商用模板：资产清单报告、漏洞整改报告、单位风险排行报告、同步质量报告。
- 模板正文支持 `{{变量名}}` 替换。
- 支持上传 `.docx` Word 模板。
- Word 模板支持变量和表格占位符。
- 支持下载和删除报表。

## 用户与权限

- 正式账号策略。
- 超级管理员、运营人员、审计员三类角色。
- 后端 API 权限控制。
- 前端菜单和路由权限控制。
- 用户创建、编辑、改密、启停。

## 审计与运维

- 登录、用户、单位、资产、漏洞、模板、报表、同步等关键操作审计。
- 健康检查和深度健康检查。
- 数据库备份、恢复、回滚。
- 部署后一键验收脚本。
